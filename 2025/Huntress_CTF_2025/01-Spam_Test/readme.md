# Spam Test - Day 01
**Category:** Warmwups  
**Author:**  [John Hammond](https://www.youtube.com/@_JohnHammond) 

## Challenge Prompt 

Time to do some careful Googling... what's the MD5 hash of the **Generic Test for Unsolicited Bulk Email** (GTUBE) string?

Submit the hash wrapped within the flag{ prefix and } suffix to match the standard flag format.

## Solution

A quick google search for “GTUBE string MD5 hash” lead to https://en.wikipedia.org/wiki/GTUBE.

The page shows the MD5 hash of the GTUBE string as:`6a684e1cdca03e6a436d182dd4069183`

Wrapped the hash in the flag format gives `flag{6a684e1cdca03e6a436d182dd4069183}`
