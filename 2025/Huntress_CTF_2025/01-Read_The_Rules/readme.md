# Read The Rules - Day 01
**Category:** Information  
**Author:**  [John Hammond](https://www.youtube.com/@_JohnHammond) 


## Challenge Prompt 
Please follow the rules for the Huntress CTF!

Read the rules at **[https://ctf.huntress.com/rules](https://ctf.huntress.com/rules)**

If you look closely, you can find a flag!

## Solution

Visit the rules page: https://ctf.huntress.com/rules
Right-click anywhere on the page and select **“View Page Source”**
Scroll through the HTML and you’ll find the flag hidden inside an HTML comment.
